# greenbite-bff

Backend For Frontend de GreenBite. Agrega datos de catálogo y pedidos para el cliente React.

## Instalación
npm install

## Ejecución
node src/index.js

## Pruebas
npm test

## Endpoints
- GET /api/dashboard/:userId — datos agregados para el dashboard
- GET /api/catalog — catálogo de cajas disponibles
